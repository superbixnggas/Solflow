# CoinFlow Solana Portfolio Optimizer - Progress Tracking

## Project Overview
Full-stack Solana portfolio optimizer with non-custodial management, auto-rebalancing, and Jupiter API integration.

## Tech Stack
- Backend: Node.js Express, Prisma ORM, PostgreSQL
- Frontend: React, TypeScript, Tailwind CSS
- APIs: Jupiter, Pyth Price, Helius, Solana RPC
- Wallets: Phantom, Backpack, Solflare
